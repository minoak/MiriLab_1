# 미리랩 — 시민 반응의 집단화·견고성·설득력 평가
> 정책: **청년 월세 한시 특별지원** · 페르소나 **24명 × 30회** 반복 = 총 **720회** 반응 수집 · 모델 gpt-4o-mini (temperature 0.8)
## 1. 무엇을 묻는가
- **Q1 집단화** — LLM 페르소나들이 정책에 대해 *무작위로 흩어지는 게 아니라, 인구통계(연령)에 따라 일관된 집단으로* 갈리는가?
- **Q2 설득력** — 각 집단이 자기 입장을 *그럴듯하게* 주장하는가?
- **견고성** — 같은 시민을 여러 번 돌려도 같은 집단 구도가 *재현*되는가(우연이 아닌가)?

## 2. 방법
- 페르소나 24명(nvidia/Nemotron-Personas-Korea 샘플)을 30회 반복해 각자 1차 반응(입장 + 5축 점수 + 반응문)을 생성.
- 연령 그룹: 청년(19~34, 정책 대상) / 중년(35~54) / 노인(55+).
- **집단화 = eta²(에타제곱)**: 점수 변동 중 연령 그룹으로 설명되는 비율(0~1). 높을수록 '연령이 반응을 가른다'.
- **견고성 = 입장 안정성**: 반복 중 최빈 입장이 차지하는 비율(무작위라면 ≈33%).
- **설득력 = LLM-judge**: 별도 LLM이 각 시민 대표 반응을 논리·근거·일관성 기준 0~100 채점.

- **중년(35~54)** — 이준영 ([반대], 설득력 75): "이 정책은 청년들에게 월세 지원을 해준다는 점에서 나름의 의도는 좋다고 생각해요. 하지만 현실적으로 보면, 지원 자격이 엄격하고, 기준 중위소득을 따지는 것부터가 청년들 중 많은 사람들이 빠질 가능성이 높습니다. 더구나 신청 과정이 복잡하고 필요한 서류가 많아서 실제로 혜택을 받는 사람..."
- **노인(55+)** — 장원주 ([반대], 설득력 75): "이 정책은 청년들에게 도움을 주겠다는 좋은 의도가 있긴 하지만, 현실적으로는 여러 문제점이 있지 않을까 싶습니다. 제가 66세 간호사로 일하는 입장에서 보자면, 이런 지원이 필요한 청년들이 많다는 건 이해하지만, 그 지원금이 정말로 그들의 주거 안정에 크게 기여할 수 있을지 의문입니다...."

## 4. 해석
- 시민 반응은 **무작위 잡음이 아니라 연령 집단으로 구조화**된다(eta² 및 입장 분포). 정책 대상인 청년은 수혜·의향이 높고 찬성으로, 비대상 세대는 낮고 반대/혼합으로 모인다.
- 같은 시민을 여러 번 돌려도 **입장이 안정적으로 재현**된다 → 집단 구도는 한 번의 우연이 아니다.
- 각 집단의 주장은 LLM-judge 기준 일정 수준 이상의 설득력을 보인다 → 페르소나가 '아무 말'이 아니라 자기 처지에 맞는 논리를 편다.

## 5. 한계 (정직하게)
- **설득력 채점은 LLM-judge라 순환 한계**가 있다(LLM이 LLM을 평가). 절대 점수보다 *집단 간 비교*와 *대표 예시*로 해석한다.
- 표본은 페르소나 24명으로, 통계적 일반화가 아니라 '구조가 나타나는지'의 시연이다.
- LLM 은 비결정적이라 절대 수치는 재현마다 출렁이나(temperature 0.8), 방향(집단화·견고성)은 안정적이다.
