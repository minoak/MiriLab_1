# 과거 정책 갭 측정 — 시뮬 − 현실 (같은 정책, 같은 단위)

- 설정: 24명 × 3회 · (dry) · temp 1.0 · 프롬프트 재설계(판단 지시 0줄 + 상세 서사 카드) 직후 베이스라인
- 정직 프레이밍: LLM이 이 정책들의 실제 여론을 학습으로 알 수 있음 — **재현 검증이지 예측력 증명 아님**.
- 강제선택(f_) = stance 우선, mixed만 lean 재배분 — 준강제선택 전화조사와 단위를 맞춘 보조 줄. 잔여(f_none) ≈ 여론조사 '모름'.

## 사전 등록 예측 (결과 보기 전 기록)

- 자격분기형(재난·월세·만나이): 분산 회복 예상. 재난 찬성 +10~20%p 과대 가능.
- 가치분기형(종부세): 이념·자산 축 부재 → 반대 소실 위험, 분열도 0.1~0.35 예상(현실 0.87). 갭 크면 처방=데이터 강화, 프롬프트 재수정 아님.

## 출처 (실측 앵커)

- 긴급재난지원금 1차 (2020): 리얼미터/YTN 2020-04-24 (지급 직전, ARS n=500 ±4.4%p). 조사방식 민감: 갤럽 면접 65 / 한국리서치 웹 54.
- 청년월세 특별지원 (2022): 전국 단위 찬반 직접 여론조사 부재 (재확인됨). 갭 측정 불가가 이 정책의 정직한 결론. 행정기록: 신청 49.5만(목표 3.3배), 심사 탈락 2/3.
- 만 나이 통일 (2023): 한국리서치 '여론 속의 여론' 2021-12 (확률표집 웹 n=1,000 ±3.1%p). 모름 14% — mixed 와 대응 비교 가능.
- 종합부동산세 강화 (2020): 리얼미터/TBS 2020-07-08 (발표기 직접 찬반, ARS n=500). 입법 직후(2020-08-12) 47.5:47.5 정확한 양분(분열도 1.00) 병기. 현실 분열의 본체 = 이념(진보 찬 74.6 vs 보수 반 65.7).

## 한계 (정직 노트)

- 표본 24명 — 비율의 최소 단위가 ~4.2%p. 분열도는 더 거칠다.
- 자극 차이: 시뮬은 발표문 전문 제공, 실측 응답자는 헤드라인 수준 정보 + 조사원 1~2문장 — 이해도·관여 방향의 체계적 차이.
- mixed ↔ '모름/무응답' 매핑은 근사(양가/모름/무관심이 한 버킷).
- 페르소나에 이념·자산(주거 점유) 축 없음 — 가치분기형 정책의 알려진 공백.
- LLM 비결정성: run 간 범위는 JSON 의 *_rng 참조.
