# 미리랩 — 정책 인생극장 (3명 대조 추적) 설계 v3 [확정]

> **사이징 확정.** v2(LLM 신경망 마을)는 3~5일엔 범위 과다 → **대조적 3명의 인생 추적**으로 확정.
> 신경망 계층은 *구현하지 않고* 슬라이드의 확장 비전으로만 쓴다.
> 이미 만든 `graph/village.py`가 본체 — 새로 만드는 게 아니라 **덜어내고 다듬는다.**
> 실제 구현은 Claude Code에서. 이 문서가 출발점.

---

## 1. 한 줄 정의

정책을 입력하면, 인구통계에 grounding된 **대조적인 3명**의 인생이 시간(턴)에 따라
어떻게 **갈리는지** 추적한다. *"같은 정책, 다른 인생."*
핵심 산출 = 차등적 영향(differential impact) + 정책 사각지대(blind spot).

## 2. 왜 3명 대조인가 (사이징 근거)

- **1명**: 핵심 메시지인 "인생이 갈린다"의 *갈림*이 안 보임.
- **24명 마을**: 범위 과다 + "대표성 있냐"는 검증 공격.
- → **3명 대조가 스윗스팟.** 한 화면에 "수혜 / 경계 / 사각지대"가 들어오고, 구현은 가볍다.
- **대조 축**: 정책 수혜 가능성이 갈리는 3명을 인구통계 기반으로 선별.
  예) 청년월세 정책 → 청년(수혜) / 중장년(경계·간접) / 고령(무관·사각지대).

## 3. 구조 (현 `village.py`가 본체)

```
정책 입력
  └─ 각 인물(3명)마다, 턴(시간 경과)마다 LLM 1회:
       place(닿은 채널) + action(행동·사건) + policy_status + economic/wellbeing + note
  └─ 누적 → 인물별 timeline (인생 궤적)
  └─ 3인 궤적을 나란히 대조
```
- **장소(채널) = 사각지대 표현 핵심** (`spaces.py` 유지): 청년은 복지로(온라인) 5분, 고령은 주민센터도 못 감 → `home`에 갇힘 = 사각지대.
- **턴 = 시간 경과**(시행 1·3·6개월 후). `advance_turn`으로 한 턴씩 진행하면 실시간 관전.
- **ablation**: `grounded=True/False`로 — grounding을 끄면 세 인물이 비슷하게 뭉개지는지 정량 비교(검증의 방패).

## 4. 덜어낸 것 (사이징 결정)

| 항목 | 처리 |
|---|---|
| 여왕 / 중간관리자 (LLM 신경망) | **구현 제외.** 슬라이드에 "3명 → 신경망으로 마을·도시 규모 확장 가능"(데자뷰)으로 *비전*만 제시 |
| 마을 전체 집계(winners/losers/blindspot_rate) | 3명엔 과함 → **개인 대조 뷰**로 단순화 |
| 정보 전파 / board / 역전파 | 3명엔 의미 약함 → 보류 |
| `graph/diffusion.py` | 폐기(삭제) |

## 6. 구현 로드맵 (Claude Code)

1. **대조 3명 선별** — 정책을 받아 수혜 가능성이 갈리는 3명을 인구통계 기반으로 고르는 로직 (`data/personas.py`)
2. **턴 단위 진행** — `simulate_village`를 `advance_turn`(한 턴씩)으로 분해 → 실시간 추적
3. **3인 대조 뷰** — 궤적 나란히 + 삶지표(economic/wellbeing) 변화 + 장소 사각지대 강조
4. **ablation** — grounding ON/OFF로 인물 개성이 사라지는지 정량화
5. *(시간 남으면)* 신경망/마을 규모 확장

## 7. 발표 포지셔닝

- **"예측"이 아니라 "영향 시나리오 / 정책 인생극장"** 으로 정직하게 포지셔닝(검증 공격 회피).
- **핵심 장면**: 같은 청년월세 정책 6개월 후 —
  A(26세 청년)는 첫 저축 / B(46세 중장년)는 자녀 위해 대신 신청 / C(74세 고령)는 정책 존재도 모름.
- **검증 = ablation**: "grounding이 이 차이를 만든다"를 수치로(인물 간 궤적 분산 ON ≫ OFF).
- **앵커**: Park GA(arXiv:2304.03442) + population-aligned(arXiv:2509.10127). RAG는 인물 기억 검색으로 확장 시 진입(옵션).
- **확장 비전**: "3명 → 데자뷰 LLM 신경망으로 마을·도시 규모까지" — 슬라이드의 미래 그림.

## 8. 범위 / 리스크

- **작게 확정.** 3명 × 3턴 ≈ 9 호출 (+ ablation 9). 데모 가볍고 빠름.
- 비결정성 → 재현성 위해 temperature↓ + seed 고정.
- Cowork 마운트 파일 쓰기 손상 관찰됨 → 로컬(Claude Code) 작업 권장.
- 잡일: `diffusion.py` 삭제.
