# 미리랩 — 정책 인생극장 (3명 대조 추적) 설계 v3 [확정]

> **v4 추가(2026-06-06): 일탈 행동 축 — §9 참고.** 시뮬의 가치 축을 '정확도(여론 재현)'에서
> '정책 스트레스 테스트(악용·편법·불만 행동화 관측)'로 넓힌다.

> **사이징 확정.** v2(LLM 신경망 마을)는 3~5일엔 범위 과다 → **대조적 3명의 인생 추적**으로 확정.
> 신경망 계층은 *구현하지 않고* 슬라이드의 확장 비전으로만 쓴다.
> 이미 만든 `graph/village.py`가 본체 — 새로 만드는 게 아니라 **덜어내고 다듬는다.**
> 실제 구현은 Claude Code에서. 이 문서가 출발점.

---

## 1. 한 줄 정의

정책을 입력하면, 인구통계에 grounding된 **대조적인 3명**의 인생이 시간(턴)에 따라
어떻게 **갈리는지** 추적한다. *"같은 정책, 다른 인생."*
핵심 산출 = 차등적 영향(differential impact) + 정책 사각지대(blind spot).

## 2. 왜 3명 대조인가 (사이징 근거)

- **1명**: 핵심 메시지인 "인생이 갈린다"의 *갈림*이 안 보임.
- **24명 마을**: 범위 과다 + "대표성 있냐"는 검증 공격.
- → **3명 대조가 스윗스팟.** 한 화면에 "수혜 / 경계 / 사각지대"가 들어오고, 구현은 가볍다.
- **대조 축**: 정책 수혜 가능성이 갈리는 3명을 인구통계 기반으로 선별.
  예) 청년월세 정책 → 청년(수혜) / 중장년(경계·간접) / 고령(무관·사각지대).

## 3. 구조 (현 `village.py`가 본체)

```
정책 입력
  └─ 각 인물(3명)마다, 턴(시간 경과)마다 LLM 1회:
       place(닿은 채널) + action(행동·사건) + policy_status + economic/wellbeing + note
  └─ 누적 → 인물별 timeline (인생 궤적)
  └─ 3인 궤적을 나란히 대조
```
- **장소(채널) = 사각지대 표현 핵심** (`spaces.py` 유지): 청년은 복지로(온라인) 5분, 고령은 주민센터도 못 감 → `home`에 갇힘 = 사각지대.
- **턴 = 시간 경과**(시행 1·3·6개월 후). `advance_turn`으로 한 턴씩 진행하면 실시간 관전.
- **ablation**: `grounded=True/False`로 — grounding을 끄면 세 인물이 비슷하게 뭉개지는지 정량 비교(검증의 방패).

## 6. 구현 로드맵 (Claude Code)

1. **대조 3명 선별** — 정책을 받아 수혜 가능성이 갈리는 3명을 인구통계 기반으로 고르는 로직 (`data/personas.py`)
2. **턴 단위 진행** — `simulate_village`를 `advance_turn`(한 턴씩)으로 분해 → 실시간 추적
3. **3인 대조 뷰** — 궤적 나란히 + 삶지표(economic/wellbeing) 변화 + 장소 사각지대 강조
4. **ablation** — grounding ON/OFF로 인물 개성이 사라지는지 정량화
5. *(시간 남으면)* 신경망/마을 규모 확장

## 7. 발표 포지셔닝

- **"예측"이 아니라 "영향 시나리오 / 정책 인생극장"** 으로 정직하게 포지셔닝(검증 공격 회피).
- **핵심 장면**: 같은 청년월세 정책 6개월 후 —
  A(26세 청년)는 첫 저축 / B(46세 중장년)는 자녀 위해 대신 신청 / C(74세 고령)는 정책 존재도 모름.
- **검증 = ablation**: "grounding이 이 차이를 만든다"를 수치로(인물 간 궤적 분산 ON ≫ OFF).
- **앵커**: Park GA(arXiv:2304.03442) + population-aligned(arXiv:2509.10127). RAG는 인물 기억 검색으로 확장 시 진입(옵션).
- **확장 비전**: "3명 → 데자뷰 LLM 신경망으로 마을·도시 규모까지" — 슬라이드의 미래 그림.

## 8. 범위 / 리스크

- **작게 확정.** 3명 × 3턴 ≈ 9 호출 (+ ablation 9). 데모 가볍고 빠름.
- 비결정성 → 재현성 위해 temperature↓ + seed 고정.
- Cowork 마운트 파일 쓰기 손상 관찰됨 → 로컬(Claude Code) 작업 권장.
- 잡일: `diffusion.py` 삭제.

## 9. 일탈 행동 축 (v4, 2026-06-06 확정)

> **문제의식**: 기존 시뮬은 '시민 분포에 따른 반응'(수용도)만 본다 — 정확도 게임은
> 여론조사를 못 이기는 구조적 한계가 있다(gap eval MAE 11~14%p). 정책입안자에게
> 더 가치 있는 질문은 **"이 정책이 어떻게 뚫리고 왜곡되는가"** — 편법·부정수급·
> 불만의 행동화. 여론조사가 답할 수 없는 질문이라 경쟁 축 자체가 다르다.
> 포지셔닝은 §7과 동일: 예측이 아니라 **레드팀 / 영향 시나리오**.

### 설계 결정 (사용자 합의)
- **범위**: 시민 측 편법(workaround) + 부정수급 시도(exploit) + 불만의 행동화
  (complain: 민원·항의·공론화). 제3자/시장 왜곡(임대인 흡수 등)은 v2 보류.
- **선정 = LLM 캐스팅**: react 전에 LLM 1회가 전체 인물 명단 + 정책을 보고
  인물별 일탈 성향 0~100 을 평가. **임계값(60) 이상만 자연 발현** — 고정 비율 없음,
  정책마다 0명일 수도 여럿일 수도. 발현 인물의 인물 카드에만 '속사정' 힌트 주입.
- **유형 = 자유 태그 + 집계용 대분류 병행**: `behavior_tag`(자유: "위장 전입 검토")
  + `behavior_class`(enum: comply/workaround/exploit/complain/inaction).
  stance(support/oppose/mixed → 입장 분포)와 동일한 기존 선례를 따른다.
- **취급 = 다른 페르소나와 동일**: 별도 탭 없음. 같은 react→interact→aggregate
  파이프라인을 흐르고, 히트맵·채팅·인생극장에 자연 등장. 표면화는 기존 화면의
  최소 확장(히트맵 배지, 대시보드 집계 줄, 개선 탭 카드+섹션)으로만.

### 구조
```
react 전 캐스팅 1회: 인물 명단+정책 → 인물별 {score, tag(자유), rationale}
  └─ score ≥ 60 발현 → 그 인물의 react 프롬프트에 [속사정] 블록 주입
react: 전원 공통으로 '속내(behavior)' 문항 추가 — 설문(survey) **뒤**에 생성돼
  5축 측정을 구조적으로 오염시키지 않음(생성 순서 = 필드 순서)
interact: 자기 속내를 '꺼낼지 감출지'만 허용 → 편법 정보의 사회 전파는 창발
인생극장: _reaction_text 에 속내 한 줄 → 빌런의 6개월 궤적이 공짜로 나옴
aggregate: behavior_counts / deviance_rate / complaint_rate (순수 파이썬 집계)
```

### 원칙 준수
- **설문 무오염**: behavior 필드는 ReactionOut 끝(생성 순서상 survey 이후).
  5축은 기존 결정론 매핑 그대로 — cluster eval 검증 자산 보존.
- **state.py add-only**: Reaction 에 behavior_class/tag/text, SimState 에 casting.
- **키리스 폴백**: mock 12인에 행동 부여(편법 1·부정 1·민원 2), 캐스팅 실패 시
  빈 캐스팅으로 우아한 강등(react 는 평소처럼 돈다).
- **정직성**: 발현은 비결정론 — `_run_behavior_eval.py` 로 발현 안정성(같은 정책
  N회에 같은 인물이 일관 발현하는가)을 측정해 리포트에 사실대로 쓴다(cluster
  eval 의 입장 안정성 82% 측정과 동일 패턴). 캐스팅 rationale 보존 = "왜 이 사람이"
  설명 가능. UI 문구는 "가능성 시나리오, 행동 예측 아님"을 명시.
- **안전 가드**: 프롬프트는 '무엇을 하려는지(의향) 수준'만 요구 — 위조·회피
  요령의 실행 매뉴얼 생성은 금지 문구로 차단. ablation(grounded=False)은
  캐스팅 자체를 건너뜀(단일 변인 실험 유지).
